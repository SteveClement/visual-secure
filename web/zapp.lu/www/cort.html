Démonstrateur

Corriger un texte luxembourgeois


Explications

Cette version est un démonstrateur qui corrige un texte court, que vous pouvez saisir ou coller dans le champ ci-dessus, et utilise le protocole HTTP pour lancer une « servlette » Java qui incorpore le serveur de correction orthographique. La version actuelle intègre désormais la « Äifeler Regel » et montre une meilleure performance que l'ancienne version « CGI ».

Il est déconseillé de corriger des textes qui contiennent des informations confidentielles puisque les données passent en clair sur le réseau. Toutefois, aucun contenu soumis par ce formulaire n'est stocké sur le serveur Cortina. Vous pouvez aussi télécharger le serveur et l'exploiter en local chez vous.

Cette version utilise des fonctionnalités avancées de JavaScript et nécessite Internet Explorer 6, Firefox 1.5 ou Opera 9.x. Les autres navigateurs supportent seulement une visualisation des erreurs sans possibilité de correction.

Vous pouvez toujours accéder les anciennes versions en cas de problèmes avec la version actuelle:

  • version sans pop-ups (version 1.2)
  • version n'utilisant que le port HTTP 80 (version 1.1)
